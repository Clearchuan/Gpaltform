# 赛博游戏厅

这是一个基于原生 HTML/CSS/JavaScript 的静态游戏平台，适合直接部署到 GitHub Pages。

## 功能特点
- 13 款本地小游戏入口
- 本地账号系统：注册、登录、游客免登、记住我、退出登录
- 本地存档：最高分、签到、皮肤、主题、音量设置都保存在 localStorage
- 霓虹赛博风格样式，支持深浅模式与多套主题皮肤
- 全局音效与音量控制

## 目录说明
- css/：全局样式与游戏通用样式
- js/：平台逻辑、界面动效、大厅渲染、音效与游戏接口
- assets/images/：预览图与背景图片
- games/：所有小游戏页面

## 部署方式
1. 将仓库上传到 GitHub。
2. 在仓库设置中开启 GitHub Pages。
3. 选择根目录即可访问。
